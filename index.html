Hi there! My name is Ben Barriage.

I am a self motivated software engineer who has been designing and creating project with code since 2017. My career began in the Robitics and Computer Vision Laboratory at Fordham University where I worked on the simulated navigation of robots based on an environmental image database.

Since then, I have worked primarily on 3D applications using ThreeJS, Unity3D, and a propriety game engine which I assisted in building. Using these tools, I have had the opportunity to work on a variety of projects including a real time 3D collision detection system, a GPU accelerated particle effect module, and a variety of content creation tools for artists/game designers.
